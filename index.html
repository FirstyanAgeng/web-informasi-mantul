Panduan Memperbaiki Font pada Microsoft Word

1. Mengubah Font di Seluruh Dokumen

Pilih Seluruh Teks:

Tekan Ctrl + A untuk memilih seluruh teks di dokumen Anda.

Pilih Font dan Ukuran:

  1. Pergi ke tab Home.
  2. Di grup Font, pilih font yang diinginkan dari daftar dropdown.
  3. Pilih ukuran font yang diinginkan dari daftar dropdown ukuran.

2. Mengatur Font Standar

Buka Kotak Dialog Font:

  1. Di tab Home, klik tombol kecil di sudut kanan bawah grup Font untuk membuka kotak dialog Font.

Pilih Font dan Ukuran:

  1. Pilih font dan ukuran yang Anda inginkan sebagai standar.

Tetapkan Sebagai Default:

  1. Klik tombol Set As Default di bagian kiri bawah kotak dialog.
  2. Pilih apakah Anda ingin mengatur font ini sebagai default hanya untuk dokumen ini atau untuk semua dokumen berbasis template Normal.
  3. Klik OK.

3. Menggunakan Gaya Teks

Pilih Teks:

Pilih teks yang ingin Anda ubah gayanya.

Gunakan Gaya Teks:

  1. Di tab Home, di grup Styles, pilih gaya teks yang diinginkan seperti Heading 1, Heading 2, Normal, dll.

Ubah dan Perbarui Gaya:

  1. Jika Anda ingin mengubah gaya yang sudah ada, klik kanan pada gaya di grup Styles dan pilih Modify.
  2. Lakukan perubahan yang diinginkan dan klik OK.

4. Mengimpor dan Menggunakan Font Eksternal

Unduh Font:

  1. Unduh font yang diinginkan dari situs web font seperti Google Fonts atau DaFont.

Instal Font:

  1. Klik dua kali pada file font yang diunduh dan klik Install untuk menginstalnya di sistem Anda.

Gunakan Font di Word:

  1. Buka kembali Word dan font yang baru diinstal akan tersedia di daftar dropdown font.

Contoh Mengubah Font

Jika Anda ingin mengubah font di seluruh dokumen menjadi Arial dengan ukuran 12, berikut adalah langkah-langkah yang lebih spesifik:

Pilih Seluruh Teks:

Tekan Ctrl + A.

Pilih Font Arial:

  1. Di tab Home, pilih Arial dari daftar dropdown font.

Pilih Ukuran 12:

  1. Pilih 12 dari daftar dropdown ukuran font.

Tetapkan Sebagai Default (Opsional):

  1. Buka kotak dialog Font (klik tombol kecil di sudut kanan bawah grup Font).
  2. Pilih Arial dan ukuran 12.
  3. Klik Set As Default, lalu pilih untuk semua dokumen berbasis template Normal.
